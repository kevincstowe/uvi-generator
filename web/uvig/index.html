Unified Verb Index Generator
The files located in this directory of the web server comprise the tool that automatically generated the PHP files in verb-index/. Those with user accounts on this server may cd to this directory and follow the instructions there for regenerating the files or modifying the tool.

However, for easy access to the Javadoc documentation for the tool, a link is provided: Javadoc Documentation.
